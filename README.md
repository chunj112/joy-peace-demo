# Joy And Peace Nails - GitHub Pages Static Build

This folder is a static HTML/CSS/JS version for client preview on GitHub Pages.

No WordPress, PHP, database, or admin login is required.

## Files to deploy

Push everything inside this `github-pages/` folder:

- `index.html`
- `services/`
- `assets/`

## GitHub Pages setup

1. Create a GitHub repository.
2. Upload the contents of `github-pages/` to the repository root.
3. Go to `Settings -> Pages`.
4. Choose `Deploy from a branch`.
5. Select branch `main` and folder `/root`.
6. Wait for GitHub to show the public demo link.

## Booking form

On this static version, the booking form opens the visitor's email app with a prefilled message. Booking management/admin is only available in the WordPress version.
